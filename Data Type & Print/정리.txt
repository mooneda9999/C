10진수 => 2진수
컴퓨터는 왜 2진법(0,1)을 사용할까?
컴퓨터는 회로에 전류가 흐르는지, 안 흐르는지를 이용하여 동작함. 전류가 흐를 때를 1, 흐르지 않을 때를 0으로 표현하는 것
====================
%d = 정수형 표현 (int)
%ld = 정수형 표현 (long)
%c = 문자형 표현 (char)
%f = 실수형 표현 (float)
%lf = 실수형 표현 (double)
%s = 문자열 표현
%o = 8진수 정수형 (int)
%x, %X = 16진수 정수형 (int)
====================
아스키코드란?
-ASCII (American Standard Code for Information Interchange, 미국 정보 교환 표준 부호)
상수란?
- 프로그램에서 값이 변경되지 않는 요소
- 리터럴(Literal) 상수와 매크로(Macro) 상수가 있음.
변수란?
- 특정 데이터를 저장하는 메모리 공간.
