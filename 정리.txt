컴퓨터에게 일 시키는 것을 프로그래밍이라 한다
C언어의 기본단위는 함수이다
C언어의 함수를 모아둔 것을 library라고 한다
<stduio.h> 표준 입/출력과 관련된 함수가 들어있는 헤더 파일

C언어의 기본 형태
====================================
#include <stdio.h>

int main()
{
  
  return 0;
}
====================================
return 0; 반환값을 반환해 주고 함수를 종료시키는 역할을 한다
void 반환값이 없다는 뜻이다
"" 문자 여러 개
'' 문자 한 개
;코드 한 줄이 끝났음을 알려주는 기호
"줄을 바꿔"라는 신호는 \n
특수문자를 출력하기 위해서는 다음과 같이 써야 합니다
\'  \\  \"  %%
